# 🎯 Table Tracker - Enhanced Billing System

A real-time table tracking and billing system built with Flask, featuring desktop and mobile interfaces for managing multiple tables with automatic time tracking and billing calculations.

![Python](https://img.shields.io/badge/python-v3.7+-blue.svg)
![Flask](https://img.shields.io/badge/flask-v2.0+-green.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

## ✨ Features

### 🖥️ **Desktop Interface**
- Real-time timer tracking for multiple tables
- Settings panel with configurable hourly rates
- Session history with scrollable data view
- Split bill functionality among multiple players
- Live updates every second
- Fixed scrolling jitter - smooth user experience

### 📱 **Mobile Interface**
- Remote control for all tables
- Responsive design for mobile devices
- Quick action buttons (Start/Pause/End)
- Recent session summaries
- Real-time synchronization with desktop

### 💰 **Billing Features**
- Flexible pricing: ₹2.0 to ₹6.5 per minute
- Automatic amount calculation
- Session history tracking
- Bill splitting among players
- Clear session data functionality

### ⚙️ **Advanced Settings**
- Rate configuration per table
- Cannot change rates during active sessions
- Persistent session data
- Export-ready session history
